# Job Roles for Technology Classification

## Complete List of Job Roles

### Development Roles
1. **frontend_developer** - Specializes in client-side web development (React, Vue, CSS, etc.)
2. **backend_developer** - Focuses on server-side development (Node.js, Python, databases, etc.)
3. **fullstack_developer** - Works on both frontend and backend technologies
4. **mobile_developer** - Develops mobile applications (React Native, Flutter, Swift, Kotlin)

### Data & AI Roles
5. **data_scientist** - Analyzes data and builds ML models (Python, R, TensorFlow, pandas)
6. **data_engineer** - Builds data pipelines and infrastructure (Spark, Airflow, Kafka)
7. **ai_ml_engineer** - Specializes in AI/ML systems (PyTorch, MLflow, deep learning)
8. **data_analyst** - Analyzes business data (SQL, Tableau, Power BI)
9. **business_intelligence_developer** - Creates BI solutions and dashboards

### Infrastructure & Operations
10. **devops_engineer** - Manages CI/CD and deployment (Docker, Kubernetes, Jenkins)
11. **site_reliability_engineer** (SRE) - Ensures system reliability and performance
12. **cloud_architect** - Designs cloud infrastructure (AWS, Azure, GCP)
13. **database_administrator** (DBA) - Manages database systems

### Quality & Security
14. **qa_engineer** - Tests software quality (manual and automated testing)
15. **test_automation_engineer** - Specializes in test automation frameworks
16. **sdet** (Software Development Engineer in Test) - Develops testing tools and frameworks
17. **security_engineer** - Implements security measures and protocols
18. **security_analyst** - Analyzes security threats and vulnerabilities
19. **penetration_tester** - Tests systems for security weaknesses

### Design & Product
20. **ux_ui_designer** - Designs user interfaces and experiences
21. **product_designer** - Overall product design and user experience
22. **graphic_designer** - Creates visual designs and graphics

### Management & Leadership
23. **project_manager** - Manages project timelines and resources
24. **product_manager** - Defines product strategy and requirements
25. **scrum_master** - Facilitates agile development processes
26. **it_manager** - Manages IT teams and resources

## Technology Mapping Examples

### Frontend Technologies
- **React, Vue.js, Angular** → `frontend_developer`, `fullstack_developer`
- **CSS, SASS, Tailwind** → `frontend_developer`, `ux_ui_designer`
- **Figma, Sketch** → `ux_ui_designer`, `product_designer`

### Backend Technologies
- **Node.js, Python, Java** → `backend_developer`, `fullstack_developer`
- **PostgreSQL, MongoDB** → `backend_developer`, `database_administrator`
- **REST API, GraphQL** → `backend_developer`, `frontend_developer`

### DevOps Technologies
- **Docker, Kubernetes** → `devops_engineer`, `site_reliability_engineer`
- **AWS, Azure, GCP** → `cloud_architect`, `devops_engineer`
- **Jenkins, GitLab CI** → `devops_engineer`

### Data/AI Technologies
- **TensorFlow, PyTorch** → `data_scientist`, `ai_ml_engineer`
- **Pandas, NumPy** → `data_scientist`, `data_analyst`
- **Spark, Airflow** → `data_engineer`
- **Tableau, Power BI** → `data_analyst`, `business_intelligence_developer`

### Mobile Technologies
- **React Native, Flutter** → `mobile_developer`
- **Swift, Kotlin** → `mobile_developer`
- **iOS, Android** → `mobile_developer`

### Testing Technologies
- **Jest, Cypress** → `qa_engineer`, `frontend_developer`
- **Selenium, Playwright** → `test_automation_engineer`, `sdet`
- **JMeter, LoadRunner** → `qa_engineer`

### Security Technologies
- **OWASP, Burp Suite** → `security_engineer`, `penetration_tester`
- **Splunk, SIEM** → `security_analyst`
- **OAuth, JWT** → `security_engineer`, `backend_developer`
